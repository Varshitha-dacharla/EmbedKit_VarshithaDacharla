# EmbedKit - Ring Buffer Assignment

## Name
Varshitha Dacharla

## Description
Implementation of an 8-byte Ring Buffer (Circular Buffer) using C99.

## Files

### ringbuf.c
Contains:
- Ring Buffer Initialization
- Write Function
- Read Function
- Count Function
- Full Check
- Empty Check
- Test Cases

## Build Instructions

gcc -Wall -std=c99 ringbuf.c -o ringbuf

## Run

Windows:
ringbuf.exe

Linux:
./ringbuf

## Features
- FIFO Data Handling
- Circular Buffer
- Full Buffer Detection
- Empty Buffer Detection
- Wrap Around Logic
